# Auction System

Desktop application built with WPF using MVVM pattern.

## Features
- User authentication (admin and regular users)
- Admin can create auctions
- Users can place bids
- Auction lasts 2 minutes
- Winner is automatically saved to database

## Technologies
- C#
- WPF
- MVVM (RelayCommand, ViewModel)
- SQL Database# auction-system
